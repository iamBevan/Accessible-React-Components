# Checkbox

Accessible `Checkbox` component that follows the [WAI-ARIA Checkbox Pattern](https://www.w3.org/TR/wai-aria-practices/#checkbox), 
which means you'll have a working dual or tri-state toggle button regardless of the type of the underlying element.
